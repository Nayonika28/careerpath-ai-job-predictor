## 📸 **Screenshot Instructions**

### **How to Capture Screenshots**

#### **Windows Method 1: Snipping Tool**
1. Press `Windows + Shift + S`
2. Select area to capture
3. Save as PNG file

#### **Windows Method 2: Print Screen**
1. Press `PrtScn` for full screen
2. Press `Alt + PrtScn` for active window
3. Paste in Paint and save

#### **Browser Method**
1. Use browser screenshot extension
2. Capture full page or specific area
3. Save in high quality

### **Screenshot Best Practices**

#### **Quality Settings**
```
✅ Resolution: 1920x1080 or higher
✅ Format: PNG (for quality) or JPG (for size)
✅ Compression: Minimal quality loss
✅ File Naming: descriptive names (e.g., "dashboard_login.png")
```

#### **Content Guidelines**
```
✅ Include browser window for context
✅ Show important UI elements
✅ Highlight key features
✅ Ensure text is readable
✅ Avoid sensitive personal data
```

#### **Organization**
```
📁 Create screenshots folder
📝 Name files descriptively
📊 Organize by feature/page
📋 Maintain screenshot log
```
